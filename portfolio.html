Alignd Client Management System

Visit Website

Currently leading development of a comprehensive client management system for healthcare practitioners that integrates with multiple external systems and medical scheme APIs including GEMS, Bonitas, and Polmed.

Integration Note: The system seamlessly connects with South African medical schemes including GEMS (Government Employees Medical Scheme), Bonitas (one of SA's largest medical schemes), and Polmed (Police Medical Scheme).

The platform serves as a centralized system for healthcare providers to manage patient data, process claims, and access real-time information from medical aids. Built with Django and Python on the backend, Vue.js on the frontend, and deployed on AWS using Docker and Kubernetes with PostGIS for spatial data management.

Django Python Vue.js AWS Docker Kubernetes PostGIS REST APIs
